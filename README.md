GREASEMETER README:

DOWNLOAD EXPOGO ON YOUR PHONE! You may need an account to run anything.

To run this project on your device, clone the repo somewhere in your computer.

Navigate to that folder in your terminal.

run: npm install
or
yarn

This should install all dependencies needed to run the project.

Now run: npx expo start


It might take a second, but a QR code should appear in your terminal. 

Use your camera to scan the QR code, this should start up the expo app, and run GreaseMeter!
